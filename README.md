# 2人の記念日カウント

スマホで使いやすいことを最優先に再設計した、**下部ナビゲーション付きの疑似マルチ画面型**記念日管理Webアプリです。  
HTML / CSS / Vanilla JavaScript だけで構成しており、外部ライブラリなしで GitHub Pages にそのまま公開できます。

## アプリ概要

「2人の名前」「付き合った日」「大切な記念日」をまとめて保存し、ホーム画面で次に近いイベントをすぐ確認できます。

主な特徴:

- スマホ最優先のアプリ風UI
- 下部固定ナビによる4画面切り替え
- ホーム画面で主要情報を1画面に集約
- 記念日一覧の検索 / カテゴリ絞り込み
- 記念日の追加 / 編集 / 削除
- JSONバックアップ / 復元
- localStorage による軽量保存
- GitHub Pages でそのまま動作

## 新しい画面構成

### 1. ホーム

ホームでは以下をまとめて表示します。

- 2人の名前
- 付き合った日
- 付き合ってから今日までの日数
- 次の月記念日まであと何日
- 次の年記念日まであと何日
- 直近の任意記念日1件
- 今日が月記念日 / 年記念日ならお祝いメッセージ

スマホで縦に長くなりすぎないよう、主要なカードをコンパクトにまとめています。

### 2. 記念日一覧

登録済みの記念日を、**近い順**に並べて表示します。

- 検索機能
- カテゴリ絞り込み
- 今日の記念日に「今日」バッジ
- 7日以内の記念日に「まもなく」バッジ
- カードはコンパクト表示
- タップで詳細展開
- 編集 / 削除ボタン付き

### 3. 追加 / 編集

追加画面は新規追加と編集の両方に対応しています。

入力項目:

- 記念日名
- 日付
- カテゴリ
- メモ

カテゴリは次の5種類です。

- デート
- 旅行
- はじめて
- プレゼント
- その他

保存後は一覧またはホームへ戻れます。

### 4. 設定

設定画面では以下を管理します。

- 2人の名前
- 付き合った日
- テーマカラー切り替え（やさしい青 / ピンク / ラベンダー）
- JSONバックアップ書き出し
- JSONバックアップ読み込み

設定はカード単位に分けて、役割がわかりやすい構成にしています。

## ローカルでの使い方

ビルドは不要です。次のどちらかで使えます。

### 方法1: ファイルを直接開く

1. このリポジトリをローカルに配置します。
2. `index.html` をブラウザで開きます。

### 方法2: 簡易ローカルサーバーで開く

例:

```bash
python3 -m http.server 8000
```

その後、ブラウザで以下を開きます。

```text
http://localhost:8000
```

## GitHub Pages 公開手順

1. このリポジトリを GitHub に push します。
2. GitHub リポジトリの **Settings** を開きます。
3. 左メニューから **Pages** を開きます。
4. **Build and deployment** で **Deploy from a branch** を選びます。
5. 公開対象ブランチを選びます。
6. フォルダは **`/ (root)`** を選びます。
7. 保存後、数分待つと公開URLが発行されます。
8. 発行されたURLにアクセスすると、このアプリがそのまま表示されます。

## localStorage 保存について

このアプリはブラウザの `localStorage` を使ってデータを保存します。  
サーバー通信は行わないため、GitHub Pages のような静的ホスティングでも動きます。

保存キー:

```text
anniversary-counter-app-v2
```

保存データ構造の概要:

```json
{
  "version": 2,
  "settings": {
    "personOne": "ゆう",
    "personTwo": "みお",
    "relationshipDate": "2024-04-01",
    "theme": "blue"
  },
  "anniversaries": [
    {
      "id": "anv_xxx",
      "title": "初デート",
      "date": "2024-04-10",
      "category": "デート",
      "note": "カフェに行った日",
      "createdAt": "2026-03-23T00:00:00.000Z"
    }
  ],
  "ui": {
    "lastScreen": "home"
  }
}
```

### 注意点

`localStorage` 保存のため、以下ではデータが消える可能性があります。

- ブラウザの閲覧データ削除
- 端末変更
- シークレットモード
- ブラウザ側のストレージ消去

必要に応じて、JSONバックアップを定期的に保存してください。

## JSONバックアップ方法

### 書き出し

1. 設定画面を開きます。
2. **「JSONを書き出す」** を押します。
3. 現在の保存データが `.json` ファイルとしてダウンロードされます。

### 復元

1. 設定画面を開きます。
2. **「JSONを読み込む」** を押します。
3. 以前書き出したバックアップJSONを選択します。
4. 読み込み後、画面表示と保存データが更新されます。

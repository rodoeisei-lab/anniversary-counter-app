# 2人の記念日カウント

GitHub Pages でそのまま公開しやすい、1ページ完結のシンプルな Web アプリです。付き合った日から今日までの日数、次の月記念日まで、次の年記念日までを自動で計算し、任意の記念日はメモ付きで一覧管理できます。

## アプリ概要

- 日本語 UI のシンプルな記念日カウントアプリ
- HTML / CSS / Vanilla JavaScript のみで構成
- スマホでも見やすいレスポンシブ対応
- 入力データはブラウザの `localStorage` に保存
- サンプルデータなしで初回表示

## ファイル構成

```text
/
├── index.html   # 画面全体の構造
├── style.css    # レイアウト・配色・レスポンシブ対応
├── script.js    # 入力処理、日付計算、localStorage 保存
└── README.md    # アプリ説明と公開手順
```

## ローカルでの開き方

1. このリポジトリをローカルに clone します。
2. ルート直下の `index.html` をブラウザで開きます。
3. `付き合った日` を入力して保存します。
4. 必要に応じて任意の記念日を名前・日付・メモ付きで追加します。

> 簡単に確認するだけなら、ダブルクリックで `index.html` を直接開いても動作します。

## GitHub Pages で公開する手順

1. このリポジトリを GitHub に push します。
2. GitHub のリポジトリ画面で **Settings** を開きます。
3. 左メニューの **Pages** を開きます。
4. **Build and deployment** の **Source** を `Deploy from a branch` にします。
5. Branch で公開したいブランチ（例: `main`）と `/ (root)` を選択して保存します。
6. 数分待つと、GitHub Pages の公開 URL が表示されます。

## 保存データについて

- 付き合った日と任意の記念日は、利用中のブラウザの `localStorage` に保存されます。
- 保存キー名は既存互換のため **`anniversary-counter-data`** のままです。
- ページを再読み込みしても、同じブラウザ・同じ端末なら入力内容が保持されます。
- ブラウザの保存データを削除すると、登録内容も消えます。
- **別端末とは自動同期されません。**

### 保存データ形式

```json
{
  "relationshipDate": "2024-01-01",
  "anniversaries": [
    {
      "id": "1710000000000-abcd",
      "name": "はじめて会った日",
      "date": "2024-02-14",
      "memo": "雨の中で待ち合わせした日"
    }
  ]
}
```

- 既存ユーザーがすでに保存している `relationshipDate` と `anniversaries` をそのまま読み込めます。
- 旧データの `anniversaries` 要素に `memo` がない場合は、アプリ内部で空文字として扱います。
- `localStorage` のキー名は変更していないため、既存データとの後方互換があります。

## 主な機能

- 付き合った日から今日までの日数表示
- 次の月記念日までのカウント表示
- 次の年記念日までのカウント表示
- 月記念日または年記念日当日の特別メッセージ表示
- 任意の記念日の追加 / 削除前確認
- 任意の記念日のメモ保存と一覧表示
- 入力バリデーション
- 月末日やうるう年を考慮した日付計算

# 2人の記念日タイムライン

「2人の記念日カウント」Webアプリを、**思い出を振り返りやすいタイムライン強化版**として作り直したバージョンです。  
HTML / CSS / Vanilla JavaScript のみで構成しており、外部ライブラリなしで GitHub Pages にそのまま配置して動かせます。

## アプリ概要

このアプリは、2人の基本情報と大切な出来事をまとめて管理できる、スマホ優先のミニアプリです。

- 2人の名前と付き合った日を保存
- 今日までの経過日数を見やすく表示
- 次の月記念日・年記念日を自動計算
- 任意記念日をカテゴリ・メモ付きで管理
- 思い出タイムラインを月ごとに整理して表示
- 近日イベントをまとめて確認
- JSONバックアップ / 復元に対応

## 主な機能

### 1. ホームダッシュボード

- 2人の名前
- 付き合った日
- 付き合ってから今日までの日数
- 次の月記念日まであと何日
- 次の年記念日まであと何日
- 直近の予定イベント
- 今日が特別な日なら強調メッセージ表示
  - 月記念日
  - 年記念日
  - 任意記念日

### 2. 近日イベント

ダッシュボード下に、今日以降の近いイベントを最大5件表示します。

- 月記念日
- 年記念日
- 任意記念日

を混在表示し、**あと何日か**を見やすく表示します。種別ごとに色味も少し変えています。

### 3. 任意記念日管理

任意記念日は以下の情報を持てます。

- 名前
- 日付
- 種別カテゴリ
- メモ
- 毎年くり返すかどうか
- 事前に目立たせる日数

また、以下の入力体験を実装しています。

- 必須項目の明示
- 入力エラーのその場表示
- 保存完了メッセージ表示
- 編集モード表示
- キャンセル操作
- 削除前確認ダイアログ

### 4. 思い出タイムライン機能

タイムラインでは、各記録に次の情報を保存できます。

- タイトル
- 日付
- 種別
- メモ
- 写真の代わりになる絵文字1つ

表示面では以下に対応しています。

- 新しい順 / 古い順の切り替え
- 月ごとの見出し表示
- 過去・未来を含む時系列表示
- 今日の出来事に「今日」バッジ
- 7日以内に来る予定に「まもなく」バッジ
- キーワード検索
- 種別フィルタ
- 未来のみ / 過去のみ / すべて の切り替え

### 5. バックアップ / 復元

- JSONエクスポート
- JSONインポート
- 不正JSONや形式不一致のエラー表示
- 復元前の確認ダイアログ

## ローカルでの使い方

1. このリポジトリを clone またはダウンロードします。
2. ルートの `index.html` をブラウザで開きます。
3. 「基本設定」で2人の名前と付き合った日を登録します。
4. 「任意記念日を追加・編集」で将来の楽しみや大切な日を登録します。
5. 「思い出タイムラインを追加・編集」で過去の思い出や予定を記録します。
6. 必要に応じて「バックアップ / 復元」からJSONを保存します。

> 開発サーバーは不要です。静的ファイルだけで利用できます。

## GitHub Pages での公開方法

1. GitHub にこのリポジトリを push します。
2. GitHub の対象リポジトリで **Settings** を開きます。
3. 左メニューの **Pages** を選択します。
4. **Build and deployment** で `Deploy from a branch` を選びます。
5. ブランチに公開したいブランチを選択します。
6. フォルダは `/ (root)` を選びます。
7. 保存後、数分待つと公開URLが発行されます。
8. 発行されたURLにアクセスすれば、そのままアプリが表示されます。

## localStorage 保存について

このアプリでは、保存データを以下の3つの概念で整理しています。

- `settings`
- `anniversaries`
- `timelineEntries`

保存先はブラウザの `localStorage` で、キーは以下です。

- `anniversary-counter-timeline-v3`

### 保存時のポイント

- 同じブラウザ・同じ端末なら再訪時にも内容を引き継げます。
- 別端末との自動同期はありません。
- ブラウザのストレージ削除で消える場合があります。
- 読み込み時は安全に正規化し、壊れたデータがあってもできるだけ破綻しにくいようにしています。

## JSONバックアップ / 復元方法

### エクスポート

1. 「JSONをエクスポート」を押します。
2. バックアップ用の `.json` ファイルが保存されます。

### インポート

1. 「JSONをインポート」を押します。
2. 以前保存した `.json` を選びます。
3. 確認ダイアログでOKを押すと復元されます。
4. 不正JSONや、必要な構造がないJSONはエラー表示されます。

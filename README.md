# ふたりカウント（毎日開きたくなる記念日カウントWebアプリ）

## 改善コンセプト
「1回登録して終わり」ではなく、**毎日開くと小さな進捗と達成感が見える**体験へ改善しました。  
トップで今日の変化・次の節目・通知予定を即表示し、操作なしでも価値が伝わる設計です。

## 今回の主な追加機能
- 今日の記念日サマリー（最も近い記念日 / 次の節目）
- カウント進行の可視化（大きなプログレスバー + カードごとのミニ進捗）
- 節目自動検出（100日 / 200日 / 365日 など）
- 疑似通知一覧（7日前 / 前日 / 当日）
- 昨日との差分表示（例: 昨日より+1日）
- 連続チェック日数の表示（毎日チェック誘導）
- 初回起動ガイド表示
- 明るい配色 + ダークモード最適化

## 技術設計（軽量・拡張しやすい）
- HTML + CSS + Vanilla JavaScript（外部依存なし）
- localStorage永続化
- 日付計算を `src/date-utils.js` に集約
- データ入出力を `src/storage.js` に分離
- 共有/画像生成を `src/share.js` に分離
- 画面制御を `src/app.js` で担当

## データ構造
```json
{
  "id": "ann_xxxxxxxx",
  "title": "付き合った日",
  "date": "2024-06-01",
  "createdAt": "2026-03-28T00:00:00.000Z",
  "message": "毎日1つずつ思い出を増やそう",
  "theme": "romantic"
}
```


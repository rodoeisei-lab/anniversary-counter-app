# ふたりカウント（共有できる記念日カードアプリ）

## 改善コンセプト
「自分で使うカウンター」から、**人に見せたくなる・そのまま共有したくなるカード体験**へ進化。  
スマホで完結できるよう、入力→表示→画像化→共有を最短導線で設計しています。

## 追加・変更した主な機能
- 記念日カードを **Canvas API で画像生成**（1080x1350）
- **Web Share API** で共有（非対応時は画像保存へフォールバック）
- 記念日ごとの **特別メッセージ** 入力
- **カウントダウン専用の全画面プレゼント表示**
- テーマ切替（シンプル / ロマンチック / ポップ）
- ダークモード対応
- 初回サンプルデータを自動表示
- 編集 / 削除をカード上から即実行

## 技術ポイント
- 静的構成（HTML / CSS / Vanilla JS）
- localStorage 永続化
- 外部 API / 外部ライブラリ依存なし
- モバイルファースト UI

## データ構造
```json
{
  "id": "ann_xxxxxxxx",
  "title": "付き合った日",
  "date": "2024-06-01",
  "message": "これからもよろしく",
  "theme": "romantic",
  "createdAt": "2026-03-27T00:00:00.000Z",
  "updatedAt": "2026-03-27T00:00:00.000Z"
}
```

## 使い方
1. 記念日フォームにタイトル・日付・メッセージ・テーマを入力して保存
2. カードの `共有` / `画像保存` / `全画面` を用途に応じて利用
3. 右下の固定 `📤 共有` ボタンで、最も近い記念日をワンタップ共有

## 将来拡張しやすい構造
- `THEMES` 定義にテンプレート追加でテーマ拡張
- `generateCardBlob()` で背景画像合成機能を追加可能

## ローカル起動
```bash
python3 -m http.server 8000
```
`http://localhost:8000` を開いてください。

## GitHub Pages 公開
1. GitHubへpush
2. Settings > Pages > Deploy from a branch
3. 対象ブランチ + `/ (root)` を指定

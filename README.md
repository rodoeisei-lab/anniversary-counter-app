# ふたりメモリー（記念日 + 予定コンパクトアプリ）

## アプリ概要
「記念日を数える」だけでなく、**次に何を楽しみにするか**をすぐ確認できるスマホ最優先Webアプリです。  
GitHub Pagesでそのまま動くように、HTML / CSS / Vanilla JavaScriptのみで構成しています。

## 主な機能
- ホームで今日の重要情報を集約表示
  - 付き合ってからの日数
  - 次の月記念日 / 年記念日までの日数
  - 次に来るイベント1件
  - 今日のひとこと
  - 今日やること候補
- 一覧画面（2タブ: 記念日 / 予定）
  - 検索
  - 種別フィルタ
  - 並び替え
  - 過去 / 未来 / すべて切り替え
  - コンパクトカード + 必要時のみ詳細展開
- クイック追加テンプレート
  - 記念日テンプレート
  - 予定テンプレート
- 記念日ごとのメモ履歴
  - 追加 / 編集 / 削除
  - 自動日付付与
  - 新しい順表示
- UX改善
  - 下部ナビ（ホーム / 一覧 / 追加 / 設定）
  - 項目近くの入力エラー
  - 保存時トースト通知
  - 削除時確認ダイアログ
  - 初回ガイドカード
  - 空データ時の案内
- バックアップ / 復元
  - JSONエクスポート
  - JSONインポート
  - データ形式チェック
  - 不正JSONエラー表示
  - 復元前確認

## 記念日機能と予定機能の違い
- 記念日
  - 思い出や節目を保存
  - 記念日ごとにメモ履歴を持てる
- 予定
  - 近い将来の行動を管理
  - 場所、準備済みチェックを持てる
  - 「今日」「もうすぐ」バッジで優先度を把握しやすい

## メモ履歴機能の説明
- 記念日の詳細を開くと、短いメモを複数登録できます。
- 各メモは作成日時を自動保存します。
- 一覧は新しい順で表示します。
- 各メモは編集・削除可能です。

## ローカル起動方法
### 1) 直接開く
`index.html` をブラウザで開くだけで動作します。

### 2) ローカルサーバーを使う
```bash
python3 -m http.server 8000
```
ブラウザで `http://localhost:8000` を開いてください。

## GitHub Pages公開方法
1. リポジトリをGitHubへpush
2. GitHubの **Settings > Pages** を開く
3. **Deploy from a branch** を選択
4. ブランチを選択（例: `main`）
5. フォルダを **/(root)** に設定
6. 保存後、発行URLへアクセス

## バックアップ / 復元方法
### エクスポート
1. 設定画面を開く
2. **JSONエクスポート** を押す
3. バックアップJSONを保存

### インポート
1. 設定画面を開く
2. **JSONインポート** を押してファイル選択
3. 復元前確認でOKを押す
4. 形式が正しければ復元

不正JSONや形式エラー時は画面上にエラーメッセージを表示します。

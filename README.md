# 2人の記念日カウント v2

スマホで使いやすいカードUIを意識して作り直した、2人のための記念日管理ミニアプリです。付き合った日からの経過日数、次の月記念日・年記念日までの日数、任意の記念日の管理を、HTML / CSS / Vanilla JavaScript だけで完結させています。

## アプリ概要

- 日本語 UI のシンプルな記念日カウント & 管理アプリです。
- GitHub Pages でそのまま公開できる、静的ファイル構成です。
- スマホで見やすいカード型レイアウトと大きな数字表示を採用しています。
- 入力したデータはブラウザの `localStorage` に保存されます。
- v2 では保存構造を見直しており、既存保存データとの互換性はありません。

## 追加した主な機能

- 2人の名前と付き合った日の保存
- ダッシュボード表示
  - 付き合ってから今日までの日数
  - 次の月記念日まであと何日
  - 次の年記念日まであと何日
  - 直近の任意記念日を強調表示
  - 今日が記念日のときのお祝いメッセージ
- 任意記念日の登録
  - 名前
  - 日付
  - カテゴリ
  - メモ
- 任意記念日の一覧管理
  - 近い順に並び替え
  - 編集
  - 削除
  - 今日バッジ
  - まもなくバッジ
- 入力バリデーション
- 削除前の確認ダイアログ
- フォーム送信後の成功メッセージ表示

## ローカルでの開き方

1. このリポジトリをダウンロード、または clone します。
2. ルート直下の `index.html` をブラウザで開きます。
3. 「基本設定」で2人の名前と付き合った日を保存します。
4. 「任意記念日を管理」から思い出を追加します。

> 開発用サーバーは不要です。`index.html` を直接開くだけで動作します。

## GitHub Pagesで公開する手順

1. このリポジトリを GitHub に push します。
2. GitHub のリポジトリ画面で **Settings** を開きます。
3. 左側の **Pages** を開きます。
4. **Source** または **Build and deployment** で `Deploy from a branch` を選びます。
5. 公開するブランチとフォルダ `/ (root)` を選んで保存します。
6. 数分待つと公開URLが発行されます。
7. 発行された URL にアクセスすると、そのままアプリを利用できます。

## 保存データについて

- データはブラウザの `localStorage` に保存されます。
- 保存キーは v2 用に **`anniversary-counter-v2`** を使用します。
- 保存内容は同じブラウザ・同じ端末でのみ利用できます。
- **別端末とは自動同期しません。**
- ブラウザのストレージを削除すると、保存内容も消えます。
- v2 は保存構造を作り直しているため、**既存保存データとの互換性はない可能性があります。**

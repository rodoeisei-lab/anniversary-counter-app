# 2人の記念日カウント v2.1

2人の大切な日を、スマホでさっと確認・記録できる日本語UIのミニアプリです。HTML / CSS / Vanilla JavaScript だけで動作し、GitHub Pages にそのまま配置して公開できます。

## アプリ概要

- 2人の名前と付き合った日を保存できる記念日カウントアプリです。
- 今日までの日数、次の月記念日、次の年記念日をダッシュボードで見やすく表示します。
- 任意の記念日をカード形式で一覧管理でき、近い順に自動で並び替えます。
- スマホ利用を前提に、押しやすいボタンと読みやすいカードUIに整えています。
- 保存データはブラウザの `localStorage` を使って保持します。

## 追加した主な機能

- 基本設定
  - 2人の名前設定
  - 付き合った日の保存
- ダッシュボード
  - 付き合ってから今日までの日数表示
  - 次の月記念日まであと何日
  - 次の年記念日まであと何日
  - 最も近い任意記念日を強調表示
  - 今日が月記念日または年記念日のときのお祝いメッセージ表示
- 任意記念日管理
  - 名前 / 日付 / カテゴリ / メモの登録
  - 編集 / 削除
  - 日付が近い順で一覧表示
  - 「今日」「まもなく」バッジ表示
- バックアップ
  - JSONファイルへの書き出し
  - JSONファイルからの復元
- 操作性改善
  - 入力バリデーション
  - 削除前の確認ダイアログ
  - 保存後の成功メッセージ表示
  - スマホで崩れにくいレイアウト

```text
/
├── index.html   # 画面全体の構造と各セクション
├── style.css    # 配色、カードUI、レスポンシブ対応
├── script.js    # 保存処理、日付計算、一覧描画、JSON入出力
└── README.md    # アプリ概要と利用方法
```

## ローカルでの開き方

1. このリポジトリをダウンロードまたは clone します。
2. ルート直下の `index.html` をブラウザで開きます。
3. 「基本設定」で2人の名前と付き合った日を保存します。
4. 「任意記念日を追加・編集」で思い出を登録します。
5. 必要なら「バックアップと復元」からJSONを書き出します。

> 開発サーバーは不要です。`index.html` を直接開くだけで使えます。

## GitHub Pagesで公開する手順

1. このリポジトリを GitHub に push します。
2. GitHub 上で対象リポジトリの **Settings** を開きます。
3. 左メニューから **Pages** を開きます。
4. **Build and deployment** で `Deploy from a branch` を選びます。
5. 公開するブランチを選び、フォルダは `/ (root)` を指定します。
6. 保存後、数分待つと公開URLが発行されます。
7. 発行されたURLにアクセスすると、そのままアプリを利用できます。

## 保存について

- 保存先はブラウザの `localStorage` です。
- 保存キーは **`anniversary-counter-v21`** です。
- 同じブラウザ・同じ端末で開いたときに前回データを引き継げます。
- **別端末とは自動同期しません。**
- ブラウザのストレージ削除やプライベートモード終了時には消える場合があります。

## JSONバックアップ / 復元方法

### 書き出し方法

1. アプリ下部の「バックアップと復元」セクションを開きます。
2. **JSONを書き出す** ボタンを押します。
3. 保存された `.json` ファイルを任意の場所に保管します。

### 復元方法

1. 「バックアップと復元」セクションの **JSONを読み込む** を押します。
2. 以前書き出した `.json` ファイルを選択します。
3. 読み込み成功メッセージが表示されたら復元完了です。

## 補足

- このアプリは静的ファイルのみで構成されており、外部ライブラリは使っていません。
- JSONバックアップを使えば、手動で別端末へデータを移すことはできます。
- ただし、クラウド保存や自動同期機能はありません。
